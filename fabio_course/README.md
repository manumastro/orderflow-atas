# Fabio Course

Fonte didattica attiva del progetto.

## Lezioni

```text
fabio1.txt
fabio2.txt
fabio3.txt
```

I file sono trascrizioni grezze delle lezioni e vanno letti integralmente e in ordine. Non costituiscono ancora una specifica software o un modello di trading.

## Mappa Del Corso

La sintesi derivata delle tre lezioni e' in:

```text
fabio-course-model-map.md
```

La mappa collega i concetti in ordine didattico e confronta il corso con i vecchi live YouTube. Non sostituisce le trascrizioni e non costituisce un contratto operativo.

## Regola Di Utilizzo

Prima di implementare un nuovo modello la sintesi deve distinguere:

```text
concetti insegnati
informazioni direttamente osservabili
passaggi discrezionali
regole potenzialmente misurabili
limiti dei dati disponibili in ATAS
```

Nessuna singola tecnica deve essere assunta come rappresentativa dell'intero corso.
