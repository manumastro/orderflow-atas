Fonte didattica attiva del progetto.

## Lezioni

```text
fabio1.txt
fabio2.txt
fabio3.txt
```

I file sono trascrizioni grezze delle lezioni e vanno letti integralmente e in ordine. Non costituiscono ancora una specifica software o un modello di trading.

## Regola Di Utilizzo

Prima di implementare un nuovo modello occorre produrre una sintesi separata che distingua:

```text
concetti insegnati
informazioni direttamente osservabili
passaggi discrezionali
regole potenzialmente misurabili
limiti dei dati disponibili in ATAS
```

Nessuna singola tecnica deve essere assunta come rappresentativa dell'intero corso.
